# Stock-Check
## Overview

Stock-Check is a web application that allows users to check real-time and historical stock data for a given stock symbol. It provides a simple and intuitive interface to fetch and display stock information, including current price, market cap, 52-week high and low, and more. The application also features a search functionality to look up stocks by name or symbol.
## Features

    Real-time Stock Data: Fetches the current stock data for a given symbol.
    Historical Stock Data: Displays a chart of historical stock prices.
    Stock Search: Allows users to search for stocks by name or symbol.
    Responsive Design: Works on both desktop and mobile devices.

## Technologies Used

    Backend: Node.js with Express
    Frontend: HTML, CSS, JavaScript
    Libraries: UIKit for styling, Chart.js for drawing charts
    API: Yahoo Finance API for fetching stock data

## How to Use

    Fetch Stock Data: Enter a stock symbol in the input field and click the "Fetch" button. The application will display the current stock data below the input field.
    View Historical Data: After fetching the stock data, a chart displaying the historical stock prices will be generated.
    Search Stocks: Enter a query in the search input field and click the "Search" button. A list of matching stocks will be displayed.

## Installation

    Clone the repository: git clone https://github.com/crowelian/stock-check.git
    Navigate to the project directory: cd stock-check
    Install dependencies: npm install
    Start the server: npm start
    Open your web browser and go to http://localhost:3138
